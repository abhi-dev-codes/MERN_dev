Who is a Frontend Developer?

A frontend developer builds the part of a website or web application that users see and interact with, focusing on the user interface (UI) and user experience (UX).
They use programming languages like HTML, CSS, and JavaScript, along with frameworks and tools, to create the visual and interactive elements of a digital product,
ensuring it is functional, accessible, and aesthetically pleasing across different devices.


Their role involves ensuring a website is not only visually appealing and functional but also responsive, accessible, and optimized for performance and user experience.

  • JavaScript
  • Tailwind CSS

  • Git and GitHub
